# Logo — Jóvenes pa Lante

## Ruta para depositar el logo

Copia el archivo del logo directamente en esta carpeta:

```
comunidad/jovenes_pa_lante/img/logo/
```

## Formatos aceptados (en orden de preferencia)

| Formato | Extensión | Cuándo usar |
|---|---|---|
| SVG | `logo_jovenes_pa_lante.svg` | Ideal — escala sin perder calidad |
| PNG con fondo transparente | `logo_jovenes_pa_lante.png` | Si no hay SVG disponible |
| PNG versión blanca | `logo_jovenes_pa_lante_blanco.png` | Para usar sobre fondos oscuros |

## Nombre de archivo esperado por la app

```
logo_jovenes_pa_lante.svg     ← principal
logo_jovenes_pa_lante.png     ← alternativo
```

Una vez copies el archivo aquí, avísale al desarrollador para integrarlo en la interfaz.
